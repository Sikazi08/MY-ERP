Analyse le fichier Excel fourni et recrée intégralement son fonctionnement sous forme d'une application ERP professionnelle moderne nommée THE HOMIES ERP.

L'objectif est de remplacer complètement Excel par une application professionnelle sécurisée, rapide, moderne, responsive et adaptée à une boutique de téléphones et accessoires.

IMPORTANT

* Reproduire fidèlement toutes les fonctionnalités du fichier Excel.
* Utiliser PostgreSQL ou Supabase comme base de données.
* Architecture prête pour la production.
* Interface moderne noir, gris foncé et orange.
* Application entièrement responsive.
* Compatible PC, tablette, Android et iPhone.
* Toutes les données doivent être synchronisées en temps réel.
